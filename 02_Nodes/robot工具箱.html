Robotics Toolbox for MATLAB 函数指南

本指南整理了 Robotics Toolbox for MATLAB 中核心函数及类的作用，帮助初学者快速了解其功能和应用场景。

🚗 导航与路径规划

1. Astar 类

  • 作用: 实现 A* 路径规划算法，用于静态环境中的最短路径搜索。
  • 特点: 支持多目标优化 (MOO) 和 Pareto 最优性 (AstarPO)。
  • 方法: plan() 计算路径，path() 生成路径，plot() 可视化地图。

2. Dstar 类

  • 作用: 动态路径规划算法，支持增量式重新规划（如环境变化时）。
  • 变种:
    • DstarMOO: 多目标优化版本。
    • DstarPO: 使用 Pareto 前沿的多目标优化。
  • 方法: modify_cost() 修改代价地图，reset() 重置规划器。

3. Bug2 类

  • 作用: 实现 Bug2 避障算法，基于局部传感器信息进行简单路径规划。
  • 适用场景: 未知或动态环境中的实时避障。

4. PRM 类

  • 作用: 概率路线图 (Probabilistic Roadmap) 算法，用于高维空间的路径规划。
  • 特点: 通过随机采样构建图结构，适合复杂环境。

🤖 机器人模型与控制

1. Bicycle 类

  • 作用: 模拟自行车模型（阿克曼转向）的车辆运动学。
  • 功能: 生成控制输入、更新状态、可视化轨迹。
  • 属性: 车辆长度 (L)、最大转向角 (alphalim)。

2. Unicycle 类

  • 作用: 差速驱动机器人模型（如两轮小车）。
  • 方法: deriv() 计算状态导数，step() 单步运动模拟。

3. SerialLink 类

  • 作用: 构建串联机械臂模型，支持运动学和动力学计算。
  • 关键方法:
    • fkine() 正运动学，ikine() 逆运动学。
    • jacob0() 计算雅可比矩阵，rne() 逆向动力学。

4. Quadrotor 类

  • 作用: 四旋翼飞行器动力学模型。
  • 功能: 模拟推力与力矩控制，生成飞行轨迹。

🔧 工具函数

1. ctraj

  • 作用: 生成笛卡尔空间中的轨迹（两点间的平滑运动）。
  • 输入: 起始和结束位姿，点数或时间向量。
  • 输出: 齐次变换矩阵序列。

2. delta2tr

  • 作用: 将微分运动（平移和旋转）转换为齐次变换矩阵。
  • 应用: 机器人末端微小运动建模。

3. trinterp

  • 作用: 在齐次变换矩阵之间插值，生成连续运动轨迹。
  • 类型: 线性或球面插值。

4. lspb

  • 作用: 生成梯形速度轨迹（Linear Segment with Parabolic Blends）。
  • 特点: 平滑加速和减速，避免速度突变。

🗺️ 地图与感知

1. LandmarkMap 类

  • 作用: 地标地图模型，用于 SLAM（同步定位与地图构建）。
  • 方法: 添加/获取地标，计算观测模型。

2. EKF 类

  • 作用: 扩展卡尔曼滤波器，用于状态估计（如机器人定位）。
  • 功能: 预测与更新步骤，支持非线性系统模型。

3. distancexform

  • 作用: 计算距离变换图，表示每个点到最近障碍物的距离。
  • 应用: 路径规划中的代价地图生成。

📐 数学与几何工具

1. SE3 类

  • 作用: 表示三维空间中的刚体变换（旋转+平移）。
  • 方法:
    • SE3.transl() 创建平移变换，SE3.rotx() 生成绕 X 轴旋转。
    • interp() 插值变换，log() 计算李代数。

2. Quaternion 类

  • 作用: 四元数表示三维旋转，避免万向节锁。
  • 方法:
    • UnitQuaternion 单位四元数，slerp() 球面线性插值。

3. Plucker 类

  • 作用: 普吕克坐标表示空间直线，用于几何计算。
  • 应用: 机器人视觉中的线特征建模。

🛠️ 实用工具

1. rtbdemo

  • 作用: 运行工具箱的演示脚本，快速查看功能示例。
  • 用法: 在 MATLAB 命令行输入 rtbdemo。

2. startup_rvc

  • 作用: 初始化工具箱路径，确保所有函数可用。
  • 注意: 安装后首次使用前需执行此脚本。

3. symexpr2slblock

  • 作用: 将符号表达式转换为 Simulink 模块，用于自定义控制算法。
